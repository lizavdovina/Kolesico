## Описание

Приложение для конвертации валют с сохранением истории операций.

## Функциональность

* Выбор валют для конвертации (из/в)
* Ввод суммы для конвертации
* Получение актуальных курсов через API
* Сохранение истории операций в JSON
* Отображение истории в таблице

## Установка и запуск

### Требования

* Python 3.6+
* Библиотеки: `requests`, `tkinter`, `json`
